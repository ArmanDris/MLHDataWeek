### Prerequisites:
Needs mongodb installed locally (https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/)
I run mongo db with:
brew services start mongodb-community@7.0
To stop mongodb:
brew services stop mongodb-community@7.0
